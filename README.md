Project Description:

The main purpose of this project is to practice Python coding by building a complete desktop application.

Cats vs Dogs TicTacToe is an advanced, strategic evolution of classical Tic-Tac-Toe designed to be an engaging alternative to simple office games like Minesweeper or Solitaire.

Key features include:
- Massive Grids: Play on elongated 15x3 or 12x4 boards, forcing you to defend multiple overlapping vertical, horizontal, and diagonal threats.
- Complex Winning Rules: Depending on the board size and direction of the line, you must secure 3, 4, or 5 consecutive pieces to win.
- Tournament Mode: Matches are 'First to 5 Wins', with the loser of the previous round earning the first-mover advantage in the next.
- AI Testing Ground: A platform to experiment with and compare the efficiency of various mathematical and statistical algorithms (Minimax, Monte Carlo Tree Search) against Generative LLMs (Llama, Claude, Gemini). A hybrid 'Centaur' system instantly secures obvious forced moves to save time, delegating complex long-term strategy to the Generative models.
